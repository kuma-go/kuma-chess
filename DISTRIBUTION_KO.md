# KUMA CHESS 1.0.0

## 웹 실행

이 폴더 전체를 HTTPS 웹 호스팅에 업로드합니다. 로컬에서는 이 폴더에서 아래 명령으로 확인할 수 있습니다.

```bash
python3 -m http.server 8005
```

브라우저에서 `http://localhost:8005/`를 엽니다.

## 모바일 설치

- Android Chrome: 게임 접속 후 브라우저 메뉴에서 `앱 설치` 또는 `홈 화면에 추가`를 선택합니다.
- iPhone Safari: 게임 접속 후 공유 버튼을 누르고 `홈 화면에 추가`를 선택합니다.
- 세로 화면과 전체 화면 실행을 기준으로 구성되어 있습니다.
- 처음 한 번 온라인으로 전체 게임을 불러오면 이후 로딩된 리소스는 오프라인 캐시에 저장됩니다.

## 저장 데이터

코인, 구매한 기물, 퍼즐 진행도와 설정은 브라우저의 로컬 저장소에 보관됩니다. 브라우저 데이터를 삭제하거나 앱을 제거하면 초기화될 수 있습니다.

## 네이티브 앱

이 배포본은 설치형 PWA입니다. Android APK/AAB와 iOS IPA/App Store 패키지는 Capacitor 래퍼와 각 플랫폼의 서명 설정을 추가해 별도로 생성할 수 있습니다.
